# Estructuras de Datos y GUI con Python

Este repositorio contiene ejercicios y proyectos de estructuras de datos y desarrollo de interfaces gráficas con Tkinter.

## Contenido

### EjU2 - Ejercicios Unidad 2
- **Ej1U201.py**: Ejercicios básicos con if-elif-else
- **Ej2U201.py**: Calculadora básica con Tkinter
- **EJ3U201.py**: Formulario de validación de datos
- **calculadora_version1.py**: Calculadora versión corregida
- **calculadora_version2.py**: Calculadora optimizada con bucles

### U1 - Unidad 1
- **cuadernoU1-05.ipynb**: Notebook de ejercicios
- **cuadernoU1-06.ipynb**: Notebook de ejercicios

### U2 - Unidad 2
- **cuadernoU2-01.ipynb**: Introducción a Tkinter y TTK
- **cuadernoU2-02.ipynb**: Ejercicios avanzados de GUI
- **cuadernoU2-03.ipynb**: Proyectos finales

## Tecnologías utilizadas
- Python 3.x
- Tkinter/TTK para interfaces gráficas
- Jupyter Notebooks
